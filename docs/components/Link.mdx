# Link

Use Link to present your design and content as clearly and efficiently as possible.

Below are the relevant attributes for all Link types.

### Sizes

The hierarchy for text size in Link -

- **H6**
- **H5**
- **H4**
- **H3**
- **H2**
- **H1**
- **Subtitle1**
- **Subtitle2**
- **Body1**
- **Body2**

```jsx live
function LinkSizes() {
  return (
    <StyledBox>
        <Link href="#" level="h1">H1</Link>
        <Link href="#" level="h2">H2</Link>
        <Link href="#" level="h3">H3</Link>
        <Link href="#" level="h4">H4</Link>
        <Link href="#" level="h5">H5</Link>
        <Link href="#" level="h6">H6</Link>
        <Link href="#" level="subtitle1">Subtitle1</Link>
        <Link href="#" level="subtitle2">Subtitle2</Link>
        <Link href="#" level="body1">Body1</Link>
        <Link href="#" level="body2">Body2</Link>
    </StyledBox>
  );
}
```

### Variants

Radios are available in 4 global variants: `solid` , `soft`, `outlined` and `plain`

```jsx live
function LinkVariants() {
  return (
    <StyledBox>
      <Link href="#" variant="solid" level="h4">H4</Link>
      <Link href="#" variant="soft" level="h4">H4</Link>
      <Link href="#" variant="outlined" level="h4">H4</Link>
      <Link href="#" level="h4">H4</Link>
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function LinkVariants() {
  return (
    <StyledBox>
      <Link color="primary" level="h4">H4</Link>
      <Link color="neutral" level="h4">H4</Link>
      <Link color="danger" level="h4">H4</Link>
      <Link color="info" level="h4">H4</Link>
      <Link color="success" level="h4">H4</Link>
      <Link color="warning" level="h4">H4</Link>
    </StyledBox>
  );
}
```

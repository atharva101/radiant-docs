# IconButton

A IconButton, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all button types.

### Sizes

Buttons are available in 3 sizes: `sm`(default), `md` and `lg`

```jsx live
function ButtonSizes() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <IconButton>
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton size="md">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton size="lg">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Variants

Buttons are available in 4 global variants: `solid` (default), `soft`, `outlined` and `plain`

```jsx live
function ButtonVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <IconButton>
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton size="md" variant="outlined">
            <ChipDelete variant="plain" />
          </IconButton>
          <IconButton size="lg" variant="soft">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton size="lg" variant="plain">
            <ChipDelete variant="plain" />
          </IconButton>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Colors

All components do support 6 color type: `primary` (default), `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function ButtonVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <IconButton color="primary">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton color="neutral">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton color="danger">
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>

```

### Disabled

IconButtons in Disabled mode-

```jsx live
function ButtonVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <IconButton color="primary" disabled onClick={() => alert("Clicked")}>
            <ChipDelete variant="plain" sx={{ color: "#fff" }} />
          </IconButton>
          <IconButton variant="outlined">
            <ChipDelete variant="plain" />
          </IconButton>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

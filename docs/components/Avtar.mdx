# Avatar

An Avtar, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all button types.

### Sizes

Buttons are available in 3 sizes: `sm`, `md` (default) and `lg`

The hierarchy for text size in button labels is:

- **H4** for large buttons
- **H5** for medium buttons
- **H6** for small buttons

```jsx live
function ButtonSizes() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Avatar size="lg">L</Avatar>
          <Avatar size="md">M</Avatar>
          <Avatar size="sm">S</Avatar>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```
### Variants

Avatars are available in 4 global variants: `solid` (default), `soft`, `outlined` and `plain`

```jsx live
function ButtonVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Avatar variant="solid"></Avatar>
          <Avatar variant="soft"></Avatar>
          <Avatar variant="outlined"></Avatar>
          <Avatar variant="plain"></Avatar>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Letter Avatars

```jsx live
function letterAvatars() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Avatar sx ={{bgcolor: 'red', color: 'white'}}> H </Avatar>
          <Avatar sx ={{bgcolor: 'cyan'}}>N</Avatar>
          <Avatar sx ={{bgcolor: 'orange', color: 'white'}}>AT</Avatar>
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

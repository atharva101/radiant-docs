# Card

A Card, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all Card types.

### Basic Card

```jsx live

function BasicCard() {

  return (
    <Card sx={{ maxWidth: 275 }}>
      <CardContent>
        <Typography sx={{ fontSize: 14 }} color="text.secondary" gutterBottom>
          Best Design Library
        </Typography>
        <Typography variant="h5" component="div">
          Hi, This is radiant.
        </Typography>
      </CardContent>
    </Card>
  );
}
```
### Outlined Card

```jsx live

function BasicCard() {
    
  return (
    <Card sx={{ maxWidth: 275 }} variant = "outlined">
      <CardContent> 
         <Typography sx={{ fontSize: 14 }} color="text.secondary" gutterBottom>
          Best Design Library
        </Typography>
        <Typography variant="h5" component="div">
          Hi, This is radiant.
        </Typography>
      </CardContent>
    </Card>
  );
}
```
